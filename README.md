# Social-Network-API

## Description

An API for a social network web application where users can share their thoughts, react to friendsâ€™ thoughts, and create a friend list. this app uses Express, Node, MongoDB and Mongoose to allow users, thoughts, friend lists and thought reactions to be created, updated and deleted.

## Installation

To run this code, simply clone this repo, open the cloned code in your favorite IDE, run `npm install` to install dependencies and then run `npm start` to start the server and ensure that MongoDB and Express are installed on your computer..
